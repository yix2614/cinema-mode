# 侧边栏控件区 (Video Controls) 设计规范

包含悬浮在右侧的 `ControlPad` (上下滑动导航) 与 `EngagementPanel` (互动按钮区)，两者被统一包裹并固定位置。

## 1. 整体布局 (FullScreenVideoControls)
- **位置**: 绝对定位在屏幕右下角区域，距屏幕底部 `36px`，距右侧边界 `20px`。
- **结构**: 垂直 Flex 容器，内部包含上方的 `ControlPad` 和下方的 `EngagementPanel`，两者间距为 `16px` (`gap-4`)。
- **滑动跟随**: 整个控制区包裹在每个视频的独立遮罩层中，在 Feed 上下滑动时，当前视频的控件会跟随视频一起滑动移出，下一个视频的控件会随新视频滑入。

---

## 2. 导航控制台 (Control Pad)
位于互动区正上方，用于上下切换视频。

- **布局定位**: 
  - 自身高度由内部元素自适应撑开。
  - **位置锁定原理**: 它的绝对物理高度是通过下方 `EngagementPanel` 的**固定高度**来兜底保持稳定的。这确保了无论下方的按钮是否被隐藏，Control Pad 永远悬浮在同一个垂直高度，不会掉落。
- **外观**:
  - 形状: 胶囊圆角 (`rounded-full`)。
  - 背景: 半透明白色 (`bg-white/10`) + `8px` 毛玻璃模糊 (`backdrop-blur`)。
  - 渲染修复: 使用 `isolate`、`overflow-hidden`、`backface-visibility: hidden` 解决毛玻璃图层重影 bug。
- **按钮 (上/下箭头)**:
  - 默认: 白色图标，透明背景。
  - 悬停 (Hover): `bg-white/20` 半透明白背景。
  - 点击 (Active): 缩小至 90% (`scale-90`)。
  - 禁用 (Disabled): 透明度 `30%`，禁止点击（位于首/尾视频时）。
  - 间距: 两个按钮之间间距 `8px` (`gap-2`)。

---

## 3. 互动按钮区 (Engagement Panel)
包含头像、关注按钮、点赞、评论、收藏、分享以及旋转的音乐唱片。

- **高度机制**: 
  - **容器固定高度**: 必须设定固定高度（大屏 `420px`，小屏 `320px`）。
  - **对齐方式**: 内部内容通过 `justify-end` 和 `mt-auto` 强制**底部对齐**。这样即使隐藏了分享和音乐按钮，剩下的点赞评论等按钮也会死死贴在最底部。
  - **溢出处理**: 支持垂直滚动 (`overflow-y-auto`) 并隐藏滚动条 (`no-scrollbar`)，以防止极端小屏幕下内容溢出。
- **元素构成**:
  - **头像**: 圆形，`2px` 白色边框，底部重叠一个红色的 `+` 关注按钮 (通过 `mt-[-12px]` 负边距实现)。
  - **操作按钮**: 包含毛玻璃圆形底底座 (`rgba(255, 255, 255, 0.13)` + `blur(8px)`) 和下方的数字文本。
  - **文本标签**: `12px` 粗体白色，带投影 (`drop-shadow-md`)，行高被严格锁定为 `16px`，移除默认 `margin-top` 以紧贴图标。
  - **音乐唱片**: 黑底半透明 (`bg-black/40`)，`2px` 半透明白边框，带持续旋转动画 (`animate-spin-slow`)。

---

## 4. 响应式规则 (断点 840px)
通过监听 `window.innerWidth < 840px`，使用 JS 动态计算并内联注入尺寸，确保跨设备的完美比例。

| 组件/元素 | 属性 | 大屏 (≥ 840px) | 小屏 (< 840px) |
| :--- | :--- | :--- | :--- |
| **Engagement Panel** | **固定容器高度** | `420px` | `320px` |
| | **操作按钮尺寸** (包含底座) | `48px` | `32px` |
| | **操作按钮图标** (内部SVG) | `20px` | `16px` |
| | **头像尺寸** | `48px` | `32px` |
| | **关注(+)按钮尺寸** | `24px` | `16px` |
| | **关注(+)图标(SVG)** | `14px` | `10px` |
| | **音乐唱片尺寸** | `48px` | `32px` |
| **Control Pad** | **整体内边距** (上下左右) | `8px` | `2px` |
| | **控制按钮尺寸** | `32px` | `28px` |
| | **控制图标(SVG)** | `16px` | `14px` |
